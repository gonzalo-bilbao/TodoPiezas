1. ¿Qué es TodoPiezas?

Aplicación multiplataforma (móvil + escritorio + web) desarrollada con Flutter para la gestión y localización de repuestos de automóviles en desguaces. Centraliza el stock de múltiples desguaces, permite buscar piezas con filtros avanzados y localiza el desguace más cercano mediante geolocalización.

7. Cómo ejecutar el proyecto

Cuenta de prueba: motor@desguace.com / password123

Backend PHP (XAMPP)

1. Instalar XAMPP → https://www.apachefriends.org
2. Iniciar Apache y MySQL desde el panel de XAMPP
3. Copiar carpeta backend/ → C:\xampp\htdocs\todopiezas\
4. Abrir phpMyAdmin → crear BD "todopiezas" → importar database/schema.sql
5. Abrir http://localhost/todopiezas/seed.php → crear datos de prueba
6. Probar: http://localhost/todopiezas/desguaces/list.php

App Flutter

cd TodoPiezas
flutter pub get
flutter run -d chrome        # Ver en navegador
flutter run -d windows       # Ver en escritorio Windows
flutter run                  # Ver en emulador/móvil Android
